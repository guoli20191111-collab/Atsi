## 二、直接竞品深度拆解(三个)

### 竞品 1:**16Personalities**(全球最大 MBTI 在线测试)

| 维度 | 描述 |
|---|---|
| **产品形态** | 国际版 MBTI 在线测试,免费 60 题,付费 49 美元解锁完整报告 |
| **用户量** | 全球访问量过亿(类比 SimilarWeb 公开数据) |
| **核心价值** | "权威 MBTI + 详细人格描述" |
| **调性** | 半专业半流行,文字温和但偏长 |
| **优势** | 心智锁定(说 MBTI = 16personalities 的几乎一半用户) |
| **劣势** | 中文版本翻译生硬,内容不本地化,移动端体验一般 |
| **商业模式** | 付费深度报告 + 职业建议 |

**ATSI 学到什么:**
- ✅ 4 字母代号是个好东西(MBTI 体系最大的"社交货币"就是 INFJ/ENTP 这种代号)— ATSI 借鉴(CTRL/PUSH)
- ✅ 报告里有"知名人物匹配"会增加趣味,ATSI V2 可以考虑加"和你同人格的代表人物"
- ❌ 不要走"权威专业"路线,这条路被 MBTI 体系锁死了

**ATSI 怎么打:**
- 反"半专业"调性,把"网红口语风"做极致
- 颗粒度从 16 拆到 30,差异化抢"我厌倦 MBTI 想换一个"心智

---

### 竞品 2:**网易云人格(网易人格测试系列)**

| 维度 | 描述 |
|---|---|
| **产品形态** | 网易云音乐内置的"性格测试"H5,定期上新 |
| **用户量** | 单次活动 UV 数百万 |
| **核心价值** | "网易品牌背书 + 高品质视觉 + 营销活动驱动" |
| **调性** | 文艺、视觉精美、有故事感 |
| **优势** | 平台流量直达 + 视觉品牌力强 |
| **劣势** | 偶发性内容,不是常态化产品,用户用完就走 |
| **商业模式** | 网易自身品牌建设,辅助音乐主业 |

**ATSI 学到什么:**
- ✅ 视觉+品牌叙事很重要(ATSI 的"平行时空"是同类叙事)
- ✅ 文艺感能短期吸睛,但用户记不住——所以 ATSI 不走纯文艺,走"网红口语 + 文艺叙事"混搭
- ❌ 不要做成"营销活动型",要做成"常驻产品"

**ATSI 怎么打:**
- 视觉做出辨识度(分享卡片是关键)
- 内容上比网易云人格更"日常 + 共鸣",而不是"美但抽离"

### 竞品 3:**各种壳测试**("测你像哪种动物/食物/花/星巴克"等)

| 维度 | 描述 |
|---|---|
| **产品形态** | 大量小红书博主/微信公众号/抖音号自制的 H5 或图文测试 |
| **用户量** | 单个壳测试 UV 几千到几十万不等,总量极大 |
| **核心价值** | 娱乐 + 社交货币,完全为了"发朋友圈" |
| **调性** | 浮夸、营销梗、视觉强烈 |
| **优势** | 病毒传播力极强,制作成本低 |
| **劣势** | 内容质量低、寿命短(2-3 周热度消退)、可信度=0 |
| **商业模式** | 引流到博主公众号/小红书 |

**ATSI 学到什么:**
- ✅ 分享卡片设计要够"赏心悦目 + 有梗",这是病毒传播的核心
- ✅ 不要小看"4 字母代号"这种社交货币——壳测试也在做(只是用"猫系/狗系"这种)
- ❌ 不要做成"3 题 + 5 个壳" 的浅层测试

**ATSI 怎么打:**
- 用"30 人格 + 15 维度"的体系感打"内容深度"
- 用"网红口语风"打"调性接地气"
- 同时拥有壳测试的传播力 + 16P 的内容深度

## 三、间接竞品速览

### MBTI 官方

- 老牌权威,但**用户已经审美疲劳**
- ATSI 不直接对抗 MBTI,而是抢"MBTI 之后玩什么"心智
- 不可低估的:MBTI 体系是 ATSI 之前的"通用语言",ATSI 解读的时候可以借力(如"如果你是 INFJ,你在 ATSI 体系里更接近沙发钉子 CHIL")

### 盖洛普 CliftonStrengths

- 34 长板体系,B 端为主(企业用)
- 颗粒度 34 vs ATSI 30 接近
- **不在同一战场**,ATSI 不进职场专业测评

### 星座/塔罗

- 文化属性强,有信众基础
- 可以**结合而非竞争**:ATSI 解读里可以放"你的星座本来应该 X,但 ATSI 测出来你是 Y,这个反差就是平行时空的有趣处"

### AI 性格分析(用 Claude/GPT 自定义提示词分析)

- 极个性化但产品化弱
- ATSI 的优势是**产品形态完整 + 视觉 + 分享卡片**——这是通用 AI 给不了的

## 六、竞品启示与 ATSI 战术(总结)

| 启示 | ATSI 战术 | 优先级 |
|---|---|---|
| MBTI 体系靠 4 字母代号建立社交货币 | ATSI 用 CTRL/PUSH 等 4 字母英文码,继续强化 | ✅ 已做 |

| 抖音对"测一测"短视频流量友好 | V2 后做"30 人格速览短视频"内容矩阵 | P1 |

## ⚠️ 自用反思

1. **竞品用户量/营收数字全是公开估算**,引用前需要补一手数据
2. **横向对比矩阵的打分主观成分大**,可能因为 PM 立场偏好导致 ATSI 显得过于有利
3. **没有跑通真实"用户在 16P 和 ATSI 之间选哪个"的对比测试**,卡位是否真的成立需要 V2 上线后验证
4. **盖洛普/塔罗等被简单划入"间接/替代"**,但盖洛普的 ToB 路径在长期来看是 ATSI 商业模式 E 的最大对手,深度分析不够
5. **没有覆盖"国外英文人格测试产品"**(如 Truity / Big5)— 国内独立开发者短期不直接竞争,但 V2 后续如果出海要补
