# 人格测试题创作风格模板（适配网红口语\+多风格混搭）

## 一、核心风格定位（必遵循）

整体基调：**网红口语化\+生活化写实\+多风格混搭**，拒绝文艺鸡汤、拒绝冗长啰嗦、拒绝过于正式，兼顾“躺平摆烂感”“直白接地气”“轻微自嘲”“随性松弛”四种核心感觉，贴合普通人真实状态，读起来有共鸣、不生硬。

核心要求：题干贴合日常场景（可抽象可写实），选项4个一组，风格差异化（覆盖“躺平摆烂”“积极自律”“外向热闹”“随性松弛”四大方向），句子长短适中，有画面感，带网红口语梗（不低俗、不烂梗）。

## 二、题干创作规范（3种适配句式，可直接套用）

### 句式1：日常场景类（最常用，贴合示例题风格）

核心：围绕“休息日、独处、情绪、日常选择”等普通人高频场景，提问直白，不绕弯，有代入感。

示例：

- 休息日不用忙碌，你会怎么安排自己？

- 没人打扰、独自宅家时，你更偏向哪种状态？

- 放假不用上班，你的日常节奏是？

### 句式2：情绪场景类（贴合“心里乱成一团”示例，共情性强）

核心：聚焦“内耗、纠结、烦躁、迷茫”等常见情绪，提问接地气，戳中普通人情绪痛点。

示例：

- 心里乱成一团，怎么都捋不清的时候，你会？

- 容易想太多、陷入内耗时，你通常会？

- 遇到烦心事没法解决，你会怎么应对？

### 句式3：社交/选择类（拓展风格，丰富题型）

核心：围绕“社交、他人评价、选择困难”等场景，延续直白口语风，贴合网红“人间清醒”感。

示例：

- 面对旁人给你贴的各种标签，你会？

- 遇到纠结难选的事情，你习惯怎么选？

- 不喜欢的人情社交，你会怎么应对？

## 三、题库结构与抽题规则（必看）

### 3.1 题库结构

- 题库共 **60 道题**，覆盖 15 个维度，每维 **4 道**：1 道锚点题（4 选项，ABCD 全覆盖）+ 3 道跟进题（3 选项）。

- 每道跟进题在创作时按维度语义"舍掉最不相关的一种风格"，保留贴合该维度的 3 个。

- 全题库中，A/B/C/D 四种风格的**累计出现次数尽量均衡**（理想：每种 ~48 次，共 195 个选项位）。

### 3.2 用户抽题规则（方案 B 抽题约束）

每次给用户随机抽 **30 道题**，约束如下：

1. **每维抽 2 道**（共 15 维 × 2 = 30 道）。

2. **锚点题占比 10%**：从 15 个维度中随机选 **3 个维度作为"今日锚点维度"**，这 3 维抽题方式 = 「锚点题 + 1 道随机跟进题」；其余 12 维抽题方式 = 「2 道随机跟进题」。
   - 用户最终拿到：3 道 4 选项 + 27 道 3 选项 ≈ 1/10 比例。

3. **风格覆盖约束（防打分偏倚）**：12 个非锚点维度抽 2 跟进题时，**不能选"舍掉同一风格"的两道**——必须保证两道跟进题在 A/B/C/D 上的并集覆盖全部 4 风格。
   - 例：维度 X 的 3 道跟进题分别舍 B / B / D，则不允许同时抽 Q1 和 Q2（都舍 B），需抽 (Q1+Q3) 或 (Q2+Q3)。

### 3.3 选项标号显示规则（前端展示规范）

- **4 选项锚点题**：标号固定为 A / B / C / D，分别对应 躺平 / 自律 / 外向 / 随性。

- **3 选项跟进题**：标号统一显示为 **A / B / C**（不出现"A/C/D"这种缺号），但 A/B/C 与"躺平/自律/外向/随性"的对应关系**因题而异**——每道题需在题库元数据中标注「显示标号 → 风格」映射，供后台打分使用。
   - 例：某题舍掉「自律」风格，剩下 躺平/外向/随性 → 显示 A=躺平、B=外向、C=随性。

每种风格的内容定义如下，确保差异化、贴合网红口语感：

### 选项A：躺平摆烂型（核心：慵懒、写实、带网红口语）

关键词：卧床、刷手机、懒得动、凑合、随缘、不折腾

创作要求：有具体画面感（如“床头吸管喝水”“被窝焊身上”），句子口语化，不刻意，带轻微自嘲。

示例：

- 一觉睡到自然醒，醒了就躺床上刷手机，喝水全靠床头吸管，能不下床就尽量不下床。

- 窝在被窝里懒得动弹，手机刷到停不下来，能在床上解决的事绝不离开床。

- 干脆摆烂，啥也不想，熬一天算一天，懒得纠结那么多。

### 选项B：积极自律型（核心：充实、有条理、不摆烂）

关键词：安排行程、运动、收拾、提升自己、规律、充实

创作要求：直白说明“做什么”，不鸡汤、不刻意拔高，贴合普通人的自律状态。

示例：

- 给自己安排满满行程，运动、收拾、培养爱好，过得很充实。

- 闲不住总找事做，把房间收拾整齐，利用空闲时间提升自己。

- 按时作息、规律吃饭，每天都有固定要做的事，不浪费时间。

### 选项C：外向热闹型（核心：爱出门、喜热闹、不宅家）

关键词：出门、探店、逛街、约朋友、热闹、烟火气

创作要求：突出“外向”特质，句子轻快，贴合年轻人喜欢的社交/外出场景。

示例：

- 喜欢外出探店逛街，喜欢热闹，不习惯长时间宅家独处。

- 约朋友出门溜达、聊天，热衷于感受外面的烟火气，不爱闷在家里。

- 到处逛街觅食，喜欢新鲜事物，受不了一个人待着的冷清。

### 选项D：随性松弛型（核心：不极端、随心情、不约束）

关键词：随心情、想躺就躺、想动就动、不刻意、松弛有度

创作要求：介于“躺平”和“自律”之间，贴合普通人“不内卷、不摆烂”的松弛状态，口语化。

示例：

- 跟着心情走，想躺就躺想动就动，不刻意约束自己。

- 偶尔躺平放松，也会起身打理日常，松紧有度不沉迷懒散。

- 随心安排，不纠结、不内耗，怎么舒服怎么来。

## 四、禁忌与注意事项（必看）

1. 禁止文艺化：不写“顺其自然、坦然接纳、坚守自我内核”等文艺句子，替换为口语化表达（如“死脑子想不到，就别硬想了”“自己过得舒服就行”）。

2. 禁止冗长：每个选项控制在1\-2句话，长短适中，有画面感但不啰嗦（参考示例题长度）。

3. 禁止烂梗：不使用低俗网络梗，可使用轻微自嘲、网红口语（如“死猪不怕开水烫”“能躺不坐”），贴合普通人表达习惯。

4. 保持差异化：4个选项必须覆盖“躺平、自律、外向、随性”四种状态，不重复、不模糊。

5. 贴合日常：题干和选项都要围绕普通人的日常场景，不脱离生活，让答题者有代入感。

## 五、完整示例题（可直接参考、仿写）

### 示例1（日常场景类）

休息日不用忙碌，你会怎么安排自己？

A\. 一觉睡到自然醒，醒了就躺床上刷手机，喝水全靠床头吸管，能不下床就尽量不下床。

B\. 给自己安排满满行程，运动、收拾、培养爱好，过得很充实。

C\. 喜欢外出探店逛街，喜欢热闹，不习惯长时间宅家独处。

D\. 跟着心情走，想躺就躺想动就动，不刻意约束自己。

### 示例2（情绪场景类）

心里乱成一团，怎么都捋不清的时候，你会？

A\. 死脑子想不到，就别硬想了

B\. 越想越钻牛角尖，非要抠明白

C\. 死猪不怕开水烫，爱咋咋地

D\. 找人唠两句，糊弄过去就算。

### 示例3（社交/选择类）

面对旁人给你贴的各种标签，你会？

A\. 默默收下，躺着刷手机，压根不当回事

B\. 坚持做自己，该干啥干啥，不被别人定义

C\. 喜欢热闹，怕被孤立，刻意迎合改变自己

D\. 有点在意，但懒得解释，随别人怎么说。
